Application Programming Lab



Introduction


Each system is connected via Ethernet network to the IBM server. The main objective of the lab is to enhance the student’s knowledge regarding their work on innovative application programs with latest technologies. Of course, this lab will make students familiar with database management systems.

Objectives


1. Students will be focused to utilize the wide range of software like Oracle, SQL to understand and learn how databases works and how queries are managed.

2. This lab also focuses on developing Software Application using Microsoft .Net Framework and understands the differences between Microsoft framework and Java Framework.

3. They are also given the opportunity to connect to Linux servers and feel Open Source computability.

4. Also focuses on the basic C Programming and hence develop small real-time applications.



Scope


1. After taking this course the students would enrich their knowledge in Databases and would be able to solve any type of Queries.

2. Feel the differences between Open Source Framework and Licensed Framework.


Research and Development Lab



Introduction


Each system is connected via Ethernet network to the main file server. This lab is completely dedicated for research work of the students and faculty members. The main objective of this lab is to motivate students to involve themselves in research works and developing innovative projects. Fresh ideas of the students and faculty members are invited and providing resources to implement it.

Objectives


The R&D Lab will undertake a complete rethinking of all aspects of data analysis, and will develop: a new generation of scalable machine learning algorithms, data management tools for large-scale and heterogeneous datasets, datacenter-friendly programming models, and an improved computational infrastructure under Ubiquitous Network.



Scope


The Researchers will be able to use powerful server machine so that the huge datasets created can be made available for computing. The Server machines are powerful enough that Research Scholars can feel the computing environments. Given varieties of data, programming-models, complex infrastructures, Researchers are ready to unleash the computing power given to them.
